# 🧪 Тестирование Auto-Pagination v6.8

## Подготовка:

1. **Перезагрузи расширение:**
   - Chrome Extensions → Reload
   - Или `Ctrl/Cmd + R` на странице `chrome://extensions`

2. **Открой Amazon Orders:**
   - https://www.amazon.com/gp/your-account/order-history
   - Убедись что ты на **первой** странице

3. **Открой DevTools:**
   - F12 или `Cmd/Ctrl + Shift + I`
   - Вкладка Console

## Тестирование:

### Шаг 1: Запуск парсинга
- Нажми на иконку расширения
- Нажми **"Parse Amazon"**

### Шаг 2: Наблюдай в консоли
Ты должен увидеть:
```
🚀 Amazon Parser v6.8 (auto-pagination with clicks)
📨 ПОЛУЧЕН ЗАПРОС НА ПАРСИНГ AMAZON
📦 Запуск парсера Amazon с авто-пагинацией
🆕 Начинаем новый цикл парсинга (4 страницы)

📄 === СТРАНИЦА 1/4 ===
📦 Найдено 10 карточек заказов на текущей странице
--- Карточка 1/10 ---
...
✅ Страница 1: найдено X заказов
💾 Сохранено X заказов

⏭️ Переходим на страницу 2...
🔄 Кликаем Next...
```

### Шаг 3: Автоматический переход
- Страница **перезагрузится автоматически**
- Ты увидишь переход на страницу 2 (`startIndex=10`)

### Шаг 4: Авто-продолжение
После загрузки страницы 2:
```
🚀 Amazon Parser v6.8 (auto-pagination with clicks)
🔄 Обнаружено активное состояние пагинации - автоматически продолжаем...
   Страница 2/4

📄 === СТРАНИЦА 2/4 ===
📦 Найдено 10 карточек заказов на текущей странице
...
✅ Страница 2: найдено Y заказов
💾 Сохранено X+Y заказов

⏭️ Переходим на страницу 3...
🔄 Кликаем Next...
```

### Шаг 5: Повторяется для страниц 3 и 4
- Процесс продолжается автоматически
- Каждая страница парсится
- Данные накапливаются

### Шаг 6: Завершение
После страницы 4:
```
🎉 ПАРСИНГ ЗАВЕРШЁН!
📊 Итого найдено: XX товаров с трек-номерами
⏱️ Время: ~10-15с
```

## ✅ Что проверить:

1. **TBA треки извлекаются:**
   - Должны быть видны `✅ TBA track found: TBAXXXXXXXXXX`

2. **Переходы работают:**
   - Должны быть 4 автоматических перехода
   - URL меняется: `startIndex=0` → `10` → `20` → `30`

3. **Данные накапливаются:**
   - Счётчик `💾 Сохранено X заказов` увеличивается

4. **Кнопка Copy работает:**
   - После завершения нажми "Copy for Google Sheets"
   - Должно скопироваться ~20-30 заказов (с 4 страниц)

## ❌ Возможные проблемы:

### Проблема: "⚠️ Кнопка Next не найдена"
**Причина:** На странице нет кнопки Next (последняя страница)
**Решение:** Это нормально, парсинг завершится автоматически

### Проблема: Парсинг зависает
**Причина:** Ошибка в скрипте
**Решение:** Открой консоль, скопируй логи, очисти состояние:
```javascript
chrome.storage.local.remove('amazonPaginationState')
```

### Проблема: Дубликаты заказов
**Причина:** Парсинг запустился дважды
**Решение:** Перезагрузи storage:
```javascript
chrome.storage.local.set({ amazonOrders: [] })
```
