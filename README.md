# 📦 Pochtoy Parsing - Order Parser Pro

**Версия:** 7.5  
**Дата:** 2025-01-10

## 🚀 Описание

Расширение для браузера Chrome, которое парсит заказы с eBay, iHerb и Amazon и экспортирует их в Google Sheets.

## ✨ Основные функции

- **🛒 eBay** - парсинг заказов с eBay
- **🏥 iHerb** - парсинг заказов с iHerb  
- **📦 Amazon** - парсинг заказов с Amazon (с трек-номерами TBA)
- **🚀 Parse All** - парсинг всех магазинов одновременно
- **📊 Export** - экспорт в CSV и Google Sheets

## 🔧 Специальные функции Amazon

### 3 варианта парсинга Amazon:

1. **📦 Amazon (1 страница)** - **РЕКОМЕНДУЕТСЯ**
   - Самая надежная версия
   - Обрабатывает только первую страницу
   - Таймауты на все операции
   - Не зависает на ошибках Amazon

2. **🔧 Debug Amazon (Simple)** - для отладки
   - Обрабатывает только первые 3 карточки
   - Подробное логирование
   - Для понимания проблем

3. **Обычный Amazon** - для всех страниц
   - Парсинг 1, 2, 3 страниц
   - Улучшенная версия с таймаутами

## 📋 Как использовать

1. **Установите расширение** в Chrome
2. **Откройте** любой из поддерживаемых сайтов
3. **Нажмите** кнопку расширения
4. **Выберите** нужный вариант парсинга
5. **Скопируйте** данные в Google Sheets

## 🐛 Решение проблем

### Amazon не парсится:
1. Попробуйте кнопку **"📦 Amazon (1 страница)"**
2. Откройте консоль браузера (F12) для логов
3. Убедитесь, что вы на странице заказов Amazon

### Parse All не работает:
1. Проверьте, что все сайты открываются
2. Дождитесь загрузки всех вкладок
3. Проверьте консоль на ошибки

### Данные не появляются:
1. Проверьте кнопку "Copy for Google Sheets"
2. Убедитесь, что парсинг завершился
3. Попробуйте экспорт CSV
