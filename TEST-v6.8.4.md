#### 1. Exit English - USPS треки (PRIORITY 1)
**Проблема:** Находит только 1 USPS трек вместо 3

**Что смотреть в консоли:**
```
--- Карточка X ---
📋 Order ID: 114-1712306-1162613
📦 Найдено X кнопок Track package  ← ВАЖНО! Сколько кнопок?

--- Посылка 1 ---
  ✅ TRACK: 9261290990091555300362
  📦 Найдено X товаров в этой посылке  ← Сколько товаров Exit English?
```

**Ожидаем:**
- `📦 Найдено 3 кнопок Track package` (если меньше - проблема в HTML)
- Если 3 кнопки → должно быть 3 разных USPS трека
- Если 1 кнопка → Amazon объединил все в одну посылку

## 🔄 Шаги:

### 1. Перезагрузи расширение
- `chrome://extensions`
- Найди "Pochtoy Parsing"
- Нажми 🔄 Reload

### 2. Вернись на первую страницу Amazon
- https://www.amazon.com/gp/your-account/order-history
- Убедись что `startIndex=0` (или нет параметра)

### 3. Очисти данные
- Открой popup расширения
- Нажми "Clear Data"

### 4. Открой консоль
- F12 → вкладка Console

### 5. Запусти парсинг
- Нажми "Parse Amazon" в popup

### 6. Анализируй логи

**Для Exit English:**
1. Найди в консоли заказ `114-1712306-1162613`
2. Посчитай `📦 Найдено X кнопок Track package`
3. Посмотри сколько USPS треков найдено
4. Скопируй ВЕСЬ лог для этого заказа

**Для Multi-Order:**
1. Найди посылку `TBA324920696386`
2. Проверь есть ли `⚠️ ВНИМАНИЕ!`
3. Посмотри Order ID для каждого товара
4. Скопируй лог

### 7. Проверь в Google Sheets
- Нажми "Copy for Google Sheets"
- Вставь в таблицу
- Найди товары с `⚠️ РАЗНЫЕ ЗАКАЗЫ` в колонке Color
