## Особенности v6.7.1

### Amazon парсинг:
- **1 страница** за раз (без пагинации)
- **Итерация по кнопкам Track** - для каждой кнопки создаётся отдельная запись
- **Первый товар из карточки** - используется для всех кнопок Track в заказе
- **Поддержка TBA** - регулярное выражение: `/(TBA\d{6,})/`
- **Поддержка USPS** - регулярное выражение: `/(9\d{21,})/`
- **БЕЗ UPS** - UPS треки отфильтрованы (были ложные срабатывания)

### Механизм работы с перчатками:
Если в заказе 3 пары перчаток с 2 кнопками Track:
- Кнопка #1 (без трека) → пропускается
- Кнопка #2 (с треком) → создаётся 1 запись с названием первого товара

**Примечание:** Это ограничение Amazon, а не баг. Amazon не связывает кнопки Track с конкретными товарами в HTML.

### Механизм работы с Exit English:
Если 3 товара "Exit English" едут в 3 разных посылках USPS:
- 3 кнопки "Track package" с разными `shipmentId`
- Каждая кнопка → отдельный USPS трек
- Создаётся 3 записи (по одной на каждую посылку)
