# Шаблоны отчетов — Справочник

## Типы отчетов

| Тип | Для кого | Источники | Частота |
|-----|---------|-----------|---------|
| Еженедельный маркетинг | Руководитель | Метрика + Директ + соцсети + CRM | Еженедельно |
| Директ-отчет | Директолог | Директ + воронка Метрики | Еженедельно |
| SMM-отчет | SMM-менеджер | Соцсети + видеометрики | Еженедельно |
| CRM-дашборд | Руководитель | CRM + Метрика (сверка целей) | Ежедневно/еженедельно |

## Формат экспорта

Markdown → PDF через:
```bash
python3 /tmp/md2pdf.py input.md output.pdf
```

## Быстрая генерация отчета

```python
# Шаблон для быстрого заполнения
from datetime import date, timedelta

today = date.today()
week_ago = today - timedelta(days=7)

print(f"# Маркетинг-отчет: Pochtoy | {week_ago} — {today}")
print(f"**Подготовлен:** {today}")
```
