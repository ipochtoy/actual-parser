# Правила доставки и запрещенные товары

> **ВАЖНО:** Перед созданием рекламы конкретных товаров ВСЕГДА проверяй этот файл.

## Запрещенные к доставке товары

### Абсолютно запрещены (не рекламировать, не выкупать)

| Категория | Примеры |
|-----------|---------|
| Бытовая техника (крупная) | Холодильники, стиральные машины, плиты |
| Электроинструменты | Перфораторы, болгарки, дрели |
| Ножи и холодное оружие | Ножи, мечи, сабли, кинжалы |
| Дроны и БПЛА | Квадрокоптеры, дроны DJI и аналоги |
| Оружие и боеприпасы | Любое огнестрельное и пневматическое оружие |
| Опасные вещества | Химикаты, кислоты, взрывчатка |
| Алкоголь и табак | Любые алкогольные напитки, сигареты |
| Наркотики | Любые наркотические вещества |
| Животные и растения | Живые животные, некоторые виды растений |
| Батареи Li-Ion > 100 Вт·ч | Крупные аккумуляторы (электровелосипеды и т.п.) |

### Ограниченно разрешены (проверить перед заказом)

| Категория | Ограничение |
|-----------|-------------|
| Мелкая бытовая техника | Вес < 10 кг, одобрено командой |
| Электроника (>$1000) | Требует доп. страховки |
| Пищевые добавки | Только сертифицированные |
| Косметика и парфюмерия | Без спиртовой основы > 70% |
| Лекарства | Только безрецептурные |
| Игрушечное оружие | Только явно помеченное как игрушки |

### Особые ограничения по направлениям

| Направление | Дополнительные запреты |
|-------------|----------------------|
| Россия | Товары под санкциями, техника > определенной мощности |
| Беларусь | Аналогично России |
| Казахстан | Электроника строго по весу |
| Все направления | Товары двойного назначения |

## Разрешено к доставке (основные категории для рекламы)

✅ Одежда и обувь
✅ Аксессуары и украшения
✅ Спортивные товары (без оружия)
✅ Книги и медиа
✅ Игрушки (без оружия)
✅ Косметика (в допустимых пределах)
✅ Мелкая электроника (наушники, чехлы, кабели)
✅ Товары для дома (посуда, текстиль)
✅ Витамины и спортивное питание (сертифицированные)
✅ Товары для животных (без живых)

## Проверка товара перед рекламой

Алгоритм проверки:
1. Входит ли товар в список абсолютно запрещенных? → **СТОП**
2. Входит ли в список ограниченных? → Проверить условия
3. Соответствует ли весу/размеру для направления? → Проверить
4. Нет санкционных рисков? → Проверить
5. **Только после прохождения всех пунктов** — создавать рекламу
